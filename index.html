FAQs

JavaScript é uma linguagem de programação de alto nivel, interpretada e de script multiplataforma, de uso geral e multiplataforma. Foi desenvolvida pela Netscape Communications Corporation em 1995 para o navegador Netscape Navigator.

TypeScript é uma linguagem de programação de alto nivel desenvolvida pela Microsoft para o desenvolvimento de aplicativos web e para a plataforma .NET.

React é uma biblioteca JavaScript de código aberto com foco em interface de usuário (UI), e uma biblioteca de componentes para construir interfaces de usuário em aplicativos web.

React Native é uma biblioteca JavaScript de código aberto com foco em interface de usuário (UI), e uma biblioteca de componentes para construir interfaces de usuário em aplicativos mobile.
